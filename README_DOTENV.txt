# Para usar dotenv-java sin Maven/Gradle:
# 1. Descarga el JAR desde: https://repo1.maven.org/maven2/io/github/cdimascio/dotenv-java/2.2.4/dotenv-java-2.2.4.jar
# 2. Coloca el archivo dotenv-java-2.2.4.jar en la carpeta lib/
# 3. Al compilar y ejecutar, agrega el JAR al classpath:
#    javac -cp "lib/dotenv-java-2.2.4.jar" -d bin src/**/*.java
#    java -cp "bin:lib/dotenv-java-2.2.4.jar" Main
